# software-development-y2

To reproduce environment:
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Or:
```bash
make env
source venv/bin/activate
```

On Windows:
```batchfile
python3 -m http.server
venv\Scripts\activate.bat
pip install -r requirements.txt
```
